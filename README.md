프롬프트 링크 : https://chatgpt.com/s/t_6a0d3b5d202c81918eb55acd8b8225a2

사용 파일 - 영상물 등급 위원회 <비디오 등급 심의 결과물. db> 
전체 파일 (최근 7년)중 최근 3개월 까지의 데이터세트로 제한. null&NaN값 삭제(수기 확인&삭제), 문장&단어 앞 뒤 불필요한 공백 삭제, 형식 통일 진행.

컬럼명 - 문자열 탐색을 통한 자동 탐색 
시각화 과정에서 오류 방지 - NULL 값 제거/NaN 값 제외/등급 선택 필터 적용. 특히 등급 선택 필터 - 사이드바 필터 기능을 이용해 원하는 등급만 선택하여 분석할 수 있도록 구현. (등급 선택에 따라 시각화 다르게 확인 가능)
문장&단어 앞 뒤 불필요한 공백 삭제 - df.columns = [str(col).strip() for col in df.columns] 코드로 일괄 삭제

데이터 = 하기와 같이 정리함. 
| 항목   | 설명                                |
| ----  | -------------------------  -       |
| 게임명  | 게임 제목                            |
| 관람등급 | 전체이용가, 12세 이용가, 청소년 이용불가 등 |
| 장르   | RPG, 액션, 스포츠 등                  |
| 출시연도 | 게임 출시 연도                       |
| 제작사  | 게임 제작 회사                        |

대시보드는 총 4개의 주요 시각화 요소로 구성.
1. 게임 등급 분포 그래프 :막대그래프를 사용하여 각 관람등급별 게임 수를 시각화하였다.등급별로 얼마나 많은 게임이 존재하는지 한눈에 확인할 수 있다.
사용 차트: Plotly Bar Chart
- 특정 연령대 대상 게임이 시장에서 얼마나 많은지
- 청소년 이용불가 게임 비율이 높은지
- 전체이용가 게임 비중이 높은지

2. 연도별 게임 등급 변화 그래프 : 라인 차트를 활용하여 연도별 게임 등급 변화 추이를 분석하였다. 각 관람등급별 게임 출시 수가 시간에 따라 어떻게 변화했는지 확인할 수 있다.
사용 차트 : Plotly Line Chart
- 특정 시기 성인 대상 게임 증가 여부
- 연령 제한 강화 추세
- 게임 산업의 변화 흐름

3. 장르별 게임 등급 히트맵
히트맵(Seaborn Heatmap) :게임 장르와 관람 등급간의 관계를 시각적으로 표현하였다. 색이 진할수록 해당 조합의 게임 수가 많다는 의미이다.
사용 차트: Seaborn Heatmap
- 어떤 장르가 특정 연령대에 집중되는지
- 장르별 등급 특성
- 시장 타겟 경향

4. 회사별 게임 비율 TOP 10 : 상위 10개 제작사의 게임 비율을 원형 그래프로 시각화하였다.
사용 차트: Plotly Pie Chart
- 특정 회사의 시장 점유율
- 대형 제작사의 영향력
- 제작사별 게임 출시 규모


[종합 결론]
본 대시보드는 게임 데이터의 관람등급/ 장르/ 출시연도/ 제작사 간 관계를 시각적으로 분석할 수 있도록 설계됨. 특히 히트맵,인터랙티브 필터,연도별 추세 분석 등을 통해 게임 산업 내 연령 타겟 변화와 시장 구조를 효과적으로 파악 가능.
또한 Streamlit 기반 웹 대시보드로 구현하여 사용자가 웹 환경에서 직접 데이터를 탐색할 수 있도록 함.
